# 救急フェア2026 AR撮影フレーム

救急フェア2026「命を守るヒーロー認定証」用のAR撮影ページ一式です。

中央の透明部分にカメラ映像が入り、周囲の認定証フレームを重ねて撮影できます。

## ファイル構成

- `index.html`
  - QRコードから開く撮影ページです。
  - カメラ起動、フレーム合成、撮影画像の保存に対応しています。
- `救急フェア2026_AR撮影フレーム_中央透明.png`
  - 実際にAR撮影で重ねる透明PNGフレームです。
- `救急フェア2026_AR撮影フレーム_透明確認用.png`
  - 透明部分を水色背景で確認するためのプレビュー画像です。
  - 公開時には必須ではありません。

## QRコードで使う手順

1. `index.html` と `救急フェア2026_AR撮影フレーム_中央透明.png` を同じフォルダ構成のままWebサーバーへアップロードします。
2. アップロード後の `index.html` のURLをQRコードにします。
3. スマートフォンでQRコードを読み取り、ブラウザでページを開きます。
4. カメラ許可を求められたら許可します。
5. 画面に被写体を合わせて `撮影` を押すと、撮影画像の保存が開始されます。

## 注意点

- カメラ機能を使うため、公開URLは原則 `https://` が必要です。
- ローカル確認時は `http://127.0.0.1` でも動作します。
- スマートフォンでは背面カメラを優先して起動します。
- ブラウザの仕様により、端末やブラウザによっては保存時に確認表示が出る場合があります。
- `index.html` とPNGフレームのファイル名を変更する場合は、`index.html` 内の画像参照も合わせて変更してください。
- SNS投稿や配布用には、撮影後に保存されたPNG画像を使用してください。

## ローカル確認

このフォルダでHTTPサーバーを起動して、以下のように確認できます。

```powershell
python -m http.server 8020 --bind 127.0.0.1
```

ブラウザで次を開きます。

```text
http://127.0.0.1:8020/index.html
```
